Storing Arrays and Objects

Arrays

Local Storage only supports strings so we have to use JSON.stringify() and JSON.parse() for arrays and objects.

You will have to look at the javaScript code to see the array and object in the code. These examples only display the results after they have been stringified to store them and then parsed to use them.

Fun Fact

similar to localStorage, there is also sessionStorage

Session storage is similar to localStorage; the only difference is while data stored in localStorage has no expiration time, data stored in sessionStorage gets cleared when the page session ends.

This example will only store the data as long as the browser is open. It will survive over reloads and restores.
