While Loop

This while loop has the user click the "ok" button while x is less than or equal to 5. When x is no longer less than or equal to 5 the loop ends.

Do While Loop

Do while Loops are almost identical to while loops in how they opperate. The difference is in the code. The while loops condition is at the start and the do while condition is at the end.

Also, notice how I put the x++; before the alert. This adds one to x before the counter making the counter start out at 1 and end with 6.

For Loop

As long as this For Loop is true( i is less than 5) it will run and have the user click "ok". When the For loop becomes False ( i equals 5) the loop ends.

For/In Loop

For/In Loop loops through the properties of an object. I set a variable as date and then put the dates properties as month: may, day: 20th, and year: 2019. The loop will run through the properties in the variable everytime the user clicks and display the next property.

For/Of Loop

While the for/in loop iterates over property names, the for/of loop iterates over property values.
